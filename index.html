VAMOS FALAR DE PYTHON?

Conhecendo seu criador!

Python é uma linguagem fácil de aprender, poderosa e versátil.Ela é multiparadigma servindo para sistemas desenvolvidos em paradigmas orientados a objetos, imperativo, orientado a eventos etc. A elegância de sintaxe e a tipagem dinâmica de Python aliadas com sua natureza interpretativa, o fazem a linguagem ideal para programas e desenvolvimento de aplicações rápidas em diversas áreas e na maioria das plataformas. A linguagem de programação, Python. Foi criada por Guido Van Rossum, um programador holandes. O nome da linguagem foi inspirado no programa de TV da BBC “Monty Python’s Flying Circus”, porque Guido Van Rossum era um grande fã do programa. Criada em 1991 com seu início de desenvolvimento em 1989 enquanto trabalhava como programador na “Centrum voor wiskunde en informatica (CWI)”,foi inicialmente como um projeto de hobby para se manter ocupado durante o Natal. Trabalhou durante mais de 7 anos na google como “staff software engineer” e um dos seus últimos trabalhos foi como “principal engineer” na Dropbox uma empresa que se estruturou desde de o início com a ajuda da linguagem python como base.

Qual a utilidade do Python?!

Python pode ser usado para diversas finalidades como:

  • Desenvolvimento web do lado do servidor: se usa muito python para sites com interações com banco de dados ou para interagir com outros sites. Também é muito útil por oferecer bibliotecas com códigos pré-prontos, facilitando o desenvolvimento.
  • Automação com scripts Python: python é muito usado para automatizar tarefas do dia a dia feitas por humanos
  • Ciência de dados e “Machine learning”: Essa é uma área muito conhecida do python, que consiste em extrair conhecimento de dados e ensinar para a máquina criando modelos de machine learning.
  • de software: O python também é muito usado para o desenvolvimento em si. podendo apoiar na manutenção dos softwares, construção automática do software, desenvolvimento de protótipos entre outras funcionalidades.
  • Automação de Testes de software: É um processo de verificar se os resultados do software estão condizentes com o esperado. Os “developers” utilizam “frameworks” de testes como Unittest, Robot e PyUnit, para testar as funções que escrevem. Os testadores também usam python para escrever seus casos de teste de vários cenários.
